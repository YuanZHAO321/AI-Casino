# 背景音乐获取建议

应用不内置任何音频（版权原因）。在「音乐」面板上传本地文件（mp3 / m4a / wav / ogg / flac）即可，支持循环与顺序播放。

## 适合高端赌场氛围的免费来源

| 来源 | 说明 |
|---|---|
| [Free Music Archive](https://freemusicarchive.org)（搜 jazz / lounge / swing，看清各曲目的 CC 协议） | 大量爵士与 Lounge，个人使用基本无碍 |
| [Incompetech](https://incompetech.com)（Kevin MacLeod，CC-BY） | `Jazz` 分类下如 *Bossa Antigua*、*Backbay Lounge*、*Lobby Time* 都是经典赌场酒廊味 |
| [Pixabay Music](https://pixabay.com/music/)（免版权） | 搜 "casino jazz" / "lounge piano" / "noir jazz" |
| [Internet Archive 78rpm 馆藏](https://archive.org/details/78rpm)（公有领域） | 上世纪三四十年代爵士原声，复古拟真感拉满 |
| 环境音：[Freesound](https://freesound.org) | 搜 "casino ambience" / "card shuffle" / "poker chips" 可叠氛围底噪 |

## 推荐风格关键词

`smooth jazz` · `lounge piano` · `bossa nova` · `big band` · `noir jazz` · `vibraphone`

## 正版歌单推荐（自行购买/订阅）

真实英国赌场（如 The Ritz Club、Hippodrome）与 Vegas 名场（Bellagio、Caesars）现场常放的风格。在 iTunes / Qobuz / 各流媒体购买或订阅后下载导入。

### 有歌词（Vocal Jazz / Rat Pack）

| 艺人/专辑 | 味道 |
|---|---|
| Frank Sinatra — *Sinatra at the Sands* | Vegas 黄金年代现场，赌城气质天花板 |
| Dean Martin — *This Time I'm Swingin'!* | Rat Pack 微醺潇洒 |
| Nat King Cole — *The Very Thought of You* | 英式酒廊的丝绒嗓 |
| Ella Fitzgerald & Louis Armstrong — *Ella and Louis* | 经典对唱，暖场神器 |
| Diana Krall — *Live in Paris* | 现代高端 Lounge 标配 |
| Michael Bublé — *Call Me Irresponsible* | 新派 Vegas 大乐队 |
| Norah Jones — *Come Away with Me* | 深夜小注慢局 |

### 无歌词（Instrumental）

| 艺人/专辑 | 味道 |
|---|---|
| Bill Evans Trio — *Waltz for Debby* | 钢琴三重奏，优雅克制 |
| Oscar Peterson Trio — *Night Train* | 摇摆律动，牌局节奏感 |
| Ahmad Jamal — *At the Pershing: But Not for Me* | 空间感极佳的酒廊钢琴 |
| Vince Guaraldi Trio — *Jazz Impressions* | 轻快不抢戏 |
| Count Basie — *April in Paris* | 大乐队铜管，开场气派 |
| *Hotel Costes* 系列合辑 | 欧陆高端会所电子 Lounge |
| *Café del Mar* / *Buddha Bar* 合辑 | 现代躺平氛围，适合长局 |

> 建议音量设在 30–50%，让牌桌声音（TTS 台词）保持在前景。
